padding - 요소의 내부여백

값의 개수별 적용

  • padding : n;양사방 동일
  • padding : n n; 상하,좌우가 동일하여 상하는 0 좌우는 50
  • padding : n n n; 상, 좌우,하 순서로 적용
  • padding : n n n n;상,우 ,하,좌 순서로 적용

polaroid (padding)

padding 의 방향별 적용 속성

  • padding top 위쪽에만 내부여백
  • padding right 오른쪽에만 내부여백
  • padding bottom 아래쪽에만 내부여백
  • padding left 왼쪽에만 내부여백
